- `internal/admin/handler.go` 完全无 auth；`huan serve --bind 0.0.0.0` 是 INDEX.md 第 64 行 advertise 的合法用法，bind 后整个 LAN 能写 content/ 任意 .md
- Admin API 路径 `POST /admin/api/content/{path}` 受 content/ 限制但可写 `.env` / `..`-style 相对路径
- 即使 bind 127.0.0.1，浏览器 CSRF 也能从任意网站 POST 到 127.0.0.1:1313（无 token、无 CORS 限制）
